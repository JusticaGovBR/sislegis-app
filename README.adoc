= Aplicação SISLEGIS

== Tecnologias
* JBoss Forge
* AngularJS
* Hibernate
* JAX-RS


== Ferramentas
* Maven
* Bower
* Grunt


== Configurar front-end

. Instalar o npm (https://www.npmjs.org/[Node Package Modules])

 $ sudo yum install npm

. Instalar o http://bower.io/[Bower]

 $ sudo npm install -g bower

. Executar o Bower dentro de src/main/webapp/

 $ bower install
